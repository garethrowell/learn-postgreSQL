     [null display is set to "NULL"]

select title, description from categories
     where description is null;

teminology - relation is a table (math set)
     tuple is a row or record

sorting with NULL
----------------

order by -> sorts NULLs last

order by mycolumn NULLS last; [gets same result]

sort with NULLS first using

order by mycolumn NULLS first;

These are defaults

order by NULLS last -> default for ASC

order by NULLS first -> default for DESC
